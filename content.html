This page builds on complex number notation to focus on arithmetic with real and imaginary parts.

Addition

To add complex numbers, add the real parts together and the imaginary parts together:

$$ (a + bi) + (c + di) = (a + c) + (b + d)i $$

Example:
$(3 + 2i) + (1 + 4i) = (3 + 1) + (2 + 4)i = 4 + 6i$

Subtraction

To subtract complex numbers, subtract the real parts and subtract the imaginary parts:

$$ (a + bi) - (c + di) = (a - c) + (b - d)i $$

Example:
$(5 + 3i) - (2 + i) = (5 - 2) + (3 - 1)i = 3 + 2i$

Multiplication

To multiply complex numbers, apply the distributive property as you would with polynomials, then simplify using the fact that $i^2 = -1$:

$$ (a + bi)(c + di) = ac + adi + bci + bdi^2 = ac + adi + bci - bd = (ac - bd) + (ad + bc)i $$

Example:
$(2 + 3i)(1 + i) = 2\cdot1 + 2\cdot i + 3i\cdot 1 + 3i\cdot i = 2 + 2i + 3i + 3i^2 = 2 + 5i + 3(-1) = 2 + 5i - 3 = -1 + 5i$
